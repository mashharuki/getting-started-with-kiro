# Getting Started with Kiro

このリポジトリは、Kiroを初めて使用する方向けの包括的なチュートリアルとハンズオンガイドです。Kiroの基本概念から実践的な使用方法まで、段階的に学習できるように構成されています。

## 学習目標

このチュートリアルを完了すると、以下のことができるようになります：

- Kiroの基本概念とスペック駆動開発の理解
- Kiroを使った実際のアプリケーション開発
- チーム開発でのKiro活用方法
- 高度な機能（MCP、Steering、Hooks）の実践的な使用

## 前提知識

- 基本的なプログラミング知識
- コマンドライン操作の基礎
- Gitの基本的な使用方法

## 推定学習時間

- **第1章**: 2-3時間（基本概念とシンプルなアプリ作成）
- **第2章**: 4-6時間（本格的なアプリ開発）
- **第3章**: 2-3時間（チーム開発の設定）

## 学習の進め方

各章は順番に進めることを推奨しますが、特定のトピックに興味がある場合は該当する章から始めることも可能です。各セクションには実践的な演習が含まれているため、実際にKiroを操作しながら学習してください。

- Kiroはkiro.devからダウンロードできます
